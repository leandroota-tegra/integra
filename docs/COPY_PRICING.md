## SECTION — “FAQ DE PLANOS”
### Título
Dúvidas sobre planos

1) **O Comece grátis dá acesso a quê?**  
Você começa pelo trial com Pagamentos para chegar ao primeiro valor rápido.

2) **Posso trocar de plano depois?**  
Sim. Você pode evoluir conforme sua necessidade.

3) **Preciso contratar todos os módulos?**  
Não. A Integra Suite é modular: ative o que fizer sentido.

4) **Como funciona o Enterprise?**  
É um plano ajustado com o time para casos com exigências específicas.

5) **Como sei se devo subir para Profissional?**  
Quando você precisar conectar frentes (processos, documentos, pessoas) e ganhar visão do todo com rastreabilidade.
