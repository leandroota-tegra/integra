## FAQ
### Eyebrow (opcional)
FAQ

### Título
Perguntas frequentes

### Itens (7)
1) **Como funciona o Comece grátis?**  
Você inicia um trial e começa por Pagamentos para configurar sua primeira cobrança recorrente.

2) **O que preciso para começar?**  
Só as informações básicas para criar a conta e configurar a cobrança. Você consegue evoluir depois.

3) **Posso cancelar a qualquer momento?**  
Sim. Você pode encerrar o trial quando quiser.

4) **Isso é só Pagamentos?**  
Não. Pagamentos é a entrada. A Integra Suite conecta outras frentes para dar visão do todo.

5) **Meus dados estão seguros?**  
Tratamos acesso e armazenamento com boas práticas e controle. (Itens específicos só se forem comprováveis.)

6) **Preciso instalar algo?**  
Não. A Integra Suite é web.

7) **Posso começar por Pagamentos e depois conectar outros módulos?**  
Sim. Essa é a lógica: comece pequeno e conecte o resto quando precisar de mais controle.
