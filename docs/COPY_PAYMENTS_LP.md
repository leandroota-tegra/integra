## FAQ
### Título
Perguntas frequentes

### Itens (8)
1) **Como funciona o Comece grátis?**  
Você inicia um trial e começa configurando Pagamentos para chegar ao primeiro valor rápido.

2) **O que preciso para começar?**  
Só o essencial para criar a conta e configurar a cobrança. Ajustes podem ser feitos depois.

3) **Posso cancelar a qualquer momento?**  
Sim. Você pode encerrar o trial quando quiser.

4) **Como funciona a cobrança recorrente?**  
Você define valor, frequência e vencimento. Depois adiciona clientes e acompanha os status.

5) **Isso é só Pagamentos?**  
Não. Pagamentos é a entrada. A suíte conecta outras frentes para dar visão do todo.

6) **Meus dados estão seguros?**  
Tratamos acesso e armazenamento com boas práticas e controle. Itens específicos só se forem comprováveis.

7) **Preciso instalar algo?**  
Não. É tudo web.

8) **Tem suporte para configurar?**  
Sim. O foco é você chegar ao primeiro valor rapidamente.
